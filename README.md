## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd threejs-elite

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Development

# Start development server with hot reload
